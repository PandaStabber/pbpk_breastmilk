Generic, Generational Pharmacokinetic Model Describing the Transport of Pollutants in Breastmilk

This is a piece of code originally developed by Dr. Tenzing Gyalpo to support a publication. It has been adapted from Matlab, made generic, and is now hosted publicly to spur further development.

Background

Biomonitoring data of persistent organic pollutants (POPs) in breast milk are increasingly collected and available for quantitative analysis of levels and time trends. A common approach is to apply log-linear regression to calculate doubling and halving times of the POP concentrations based on the temporal trend observed in breast milk. However, there are different, sometimes conflicting interpretations of these doubling and halving times. Here we present a mechanistic one-box modeling approach to generically represent the transfer of POPs from mother to child in each generation.

The publication on which the model is based is Recommendations for Evaluating Temporal Trends of Persistent Organic Pollutants in Breast Milk.

The provided APIs

WUsing the developed API in python, one is able to represent variable generational pollutant dynamics with a high degree of control. An example file is included that goes over how to use and explore the basic functions of the model in the examples folder. Searchable API documentation is also included.
